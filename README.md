# 🐝 Beecrowd Solutions - C# (.NET)

Este repositório contém as minhas resoluções para os desafios da plataforma [Beecrowd](https://www.beecrowd.com.br/), focadas em lógica de programação e algoritmos utilizando **C#**.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** C#
* **Framework:** .NET 10.0 (ou a versão que estiver usando)
* **IDE:** Visual Studio / VS Code

## 📌 Estrutura do Projeto
Cada exercício está organizado para facilitar a leitura e o teste:
* `Program.cs`: Contém a lógica principal da solução.
* Estruturas de dados otimizadas.
